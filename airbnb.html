Airbnb Analysis for Vancouver, BC

This project takes on the challenge of whether it is possible to predict the rental price of a property within $25 based on features available from Airbnb data.

Because this is a hard problem and, for any user of the predictions, requires making judgements on the basis of uncertain results, we employ methods from IBM's Trusted AI software toolkit UQ 360 to use uncertainty quantification to adequately manage decisions under conditions of uncertainty. And updated version of the Jupyter notebook below that incorporates IBM U360's methods is coming soon.

Although the dataset we focus on is Airbnb rental data from Vancouver, BC, Canada, the code is designed to be able to ingest data automatically for any city with similarly formatted data and provides tools for examining and modeling the data.

The dataset for this project was obtained from http://insideairbnb.com. The source for the UQ 360 toolkit is https://github.com/IBM/UQ360.
